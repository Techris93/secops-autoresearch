Get Started in 5 Minutes

1. Install with one command

$ curl -fsSL https://get.secopsai.dev/setup.sh | sh

The setup script validates prerequisites, creates a Python environment, and configures optional features interactively.

2. Generate a benchmark corpus

$ python generate_openclaw_attack_mix.py --stats

Creates 80 labeled events (58 benign + 22 attacks) for reproducible validation.

3. Evaluate detection accuracy

$ python evaluate.py --labeled data/openclaw/replay/labeled/attack_mix.json --mode benchmark

Runs all detection rules and reports F1 score, precision, recall, and per-rule breakdown.

Result: F1 1.0 ✓
